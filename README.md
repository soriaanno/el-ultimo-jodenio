

Este proyecto es un videojuego **First Person Shooter (FPS)** desarrollado en Unity como trabajo del curso. El objetivo principal fue crear una experiencia jugable en primera persona implementando mecánicas básicas de combate, movimiento e interacción.

## 🕹️ Características

* Movimiento en primera persona (WASD + ratón)
* Sistema de disparo
* Detección de impactos
* Enemigos básicos con comportamiento simple
* Sistema de vida/daño
* Sistema de powerups
* Interfaz básica (HUD)

## 🛠️ Tecnologías utilizadas

* Unity
* C#
* Desarrollo 3D

## 🎯 Objetivo del proyecto

Este proyecto se centró en aplicar conocimientos clave del desarrollo de videojuegos en 3D:

* Control de cámara en primera persona
* Programación de mecánicas de disparo
* Gestión de físicas y colisiones
* Implementación de lógica de enemigos
* Diseño personalizado de lore 

## 🚀 Cómo jugar

1. Ejecuta el juego
2. Muévete con el teclado (W, A, S, D)
3. Apunta con el ratón
4. Dispara y elimina a los enemigos
5. Suma los mayores puntos de baja posibles

## 📚 Aprendizaje

El desarrollo de este proyecto permitió consolidar habilidades en Unity, especialmente en la creación de juegos en primera persona y en la implementación de sistemas básicos de combate.
